# VAULT_2 Project Enjeux

Business goal:
- Reduce repeated agent errors by operationalizing reusable memory.

Operational goals:
- Force lesson capture per completed ticket.
- Retrieve only relevant memory slices for active context.
- Improve handoff quality through memory-grounded context.

Success indicators:
- Fewer repeated incidents.
- Faster onboarding of new agents.
- Better first-pass ticket quality.
